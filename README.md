# Chatbot API

This is project is a Chatbot API.

The Chatbot uses the Google Gemini API, and can be tested with this UI: https://github.com/AndreNeves97/chatbot-angular


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
